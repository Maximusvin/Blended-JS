Module 6: DOM

Mortgage

Створіть застосунок в якому буде відображатись список створених клієнтом банків для можливості взяття іпотечного кредиту. При клікові на назву банку - буде відображатись його інформація, а саме:

  • Назва банку
  • Річна відсоткова ставка по іпотеці (interest rate, %)
  • Максимальна сума кредиту, яку надає банк (Max loan, $)
  • Мінімальний платіж (Min payment, $)
  • Термін кредиту (Loan term)

Якщо доданих банків не має - показувати заглушку з текстом: “списку банків ще не має і потрібно їх створити”. Для цього потрібно клікнути по кнопці “Create a new bank”.

Якщо банки в списку є - то відображати інформацію в слідуючому порядку:

  • Порядковий номер
  • Назва банку
  • Кнопка по редагуванню
  • Кнопка по видаленню конкретного банка
  • Кнопка для створення нового банку. Має бути окремо одна на весь блок

Функції:

  • Рендер списку має здійснювати функція renderList()
  • Рендер інформації по окремому банку - функція renderInfo()
  • Розмітку інформаційного блоку - функція infoMarkup()

Якщо в списку більше 3 банків - показувати кнопку по очистці всього списку. А також - поле для пошуку/фільтрації потрібного банку

Ваша початкова html-розмітка має складатись лише з div #root. Це контейнер, куди буде рендеритись уся майбутня розмітка.

Дизайн довільний. Можете обговорити його із командою. Але стилі повинні бути написані в окремому файлі.
